# Cloze Plus介绍

Cloze Plus 是一个用于 Obsidian 的填空学习插件。它在 Cloze 填空功能的基础上，加入了学习模式、复习模式、记忆曲线复习模式，以及对表格内 Cloze 的兼容支持。

> 当前版本：`1.0`  
> 适用场景：在 Obsidian 笔记中通过隐藏关键词、点击回忆、错误统计和间隔复习来辅助学习。

## 功能简介

Cloze Plus 支持将笔记中的特定文本转换为可交互的填空内容，并提供多种学习和复习模式。

目前支持的主要功能包括：

- Cloze 填空隐藏与显示
- 学习模式
- 通篇复习模式
- 记忆曲线复习模式
- 类 Anki FSRS 的复习调度
- 表格内 Cloze 支持
- 浮动复习面板
- 点击跳转到对应 Cloze
- 错误次数统计
- 当前文件复习数据持久化

---

## 支持的 Cloze 来源

插件可根据设置，将以下内容识别为 Cloze：

- `{{花括号内容}}`
- `==高亮内容==`
- `**粗体内容**`
- `<u>下划线内容</u>`
- `*斜体内容*`
- `[括号内容]`

具体哪些语法会被识别，可以在插件设置中开启或关闭。

## 模式说明

### 1. 学习模式

学习模式用于首次学习或逐段回忆。

进入学习模式后：

- 页面中的 Cloze 会被隐藏
- 可以点击隐藏内容来显示答案
- 点击 Cloze 后会弹出一个小窗口
- 小窗口包含：
  - `✔`：记住了
  - `✘`：没记住
  - `➕`：加入记忆曲线复习

学习模式还支持一条固定的学习线：

- 学习线以下的 Cloze 默认隐藏
- 滚动到学习线以上后自动显示
- 适合边阅读边回忆

---

### 2. 通篇复习模式

通篇复习模式用于复习学习模式中标记过错误的内容。

进入通篇复习模式后，会出现一个浮动复习面板。

面板支持：

- 查看当前文件中未掌握的 Cloze
- 按错误次数筛选
- 点击条目跳转到对应位置
- 上一个 / 下一个导航
- 清空当前文件复习记录

错误次数会影响 Cloze 的颜色显示，用于区分掌握程度。

---

### 3. 记忆曲线复习模式

记忆曲线复习模式用于对加入复习队列的 Cloze 进行间隔复习。

当你在学习模式中点击 `➕` 后，该 Cloze 会加入记忆曲线复习。

进入记忆曲线复习模式后，可以按以下条件筛选：

- 所有复习
- 本日到期 + 过期
- 本日到期
- 过期

点击 Cloze 后会出现评分按钮：

- 重来
- 较难
- 记得
- 简单

插件会根据评分更新该 Cloze 的复习状态和下次复习时间。

## FSRS 记忆曲线

本插件尝试接入类似 Anki FSRS 的间隔复习逻辑，用于根据用户评分调整复习间隔。

每个加入记忆曲线的 Cloze 会保存类似以下状态：

- 稳定性
- 难度
- 复习次数
- 遗忘次数
- 上次复习时间
- 下次到期时间
- 最近一次评分

注意：

> 当前实现由 AI 生成，并不保证与官方 Anki FSRS 完全一致。  
> 如果你对 FSRS 算法精度有严格要求，请自行审查代码。

## 表格支持

Cloze Plus 对 Obsidian 实时预览中的 Markdown 表格做了专门适配。

支持在表格单元格中使用：

插件会尽量只遮挡真正的 Cloze 内容，而不是整个单元格。

当前表格支持已经处理了一些复杂情况：

- 同一个单元格多个 Cloze
- 同一个单元格内重复文本
- <br> 换行
- 不同表格中存在相同文本
- 表格内 Cloze 定位跳转
- 但复杂表格仍可能存在显示偏移或性能问题。

## 数据存储说明

插件支持两种数据组织方式：

### 1. 单一 data
所有复习数据保存在一个文件中。

### 2. 每文件独立 json
每篇笔记单独保存一个 json，方便同步与查看。

插件还支持两种保存位置：

### 插件私有目录
数据保存在插件自己的数据位置中。

### 库内文件夹
数据保存在当前 Obsidian 仓库（vault）内的普通文件夹中。
适合与导出目录放在一起，也适合同步、备份、版本管理。

## 标签说明

### 作用标签
用于控制插件是否对当前笔记生效。

例如设置为：

```txt
#cloze
```

表示只有带有 `#cloze` 标签的笔记才会启用本插件处理。

### 导出标签
用于导出卡片时附加到卡片上的标签。

例如：

```txt
clozeplus 法考 安全生产
```

它不会影响插件是否生效，只影响导出结果中的标签内容。

## 安装方法

### 手动安装

1. 下载本仓库中的插件文件：
   - `main.js`
   - `manifest.json`
   - `styles.css`

2. 在你的 Obsidian 库中创建插件目录：

```text
.obsidian/plugins/cloze-plus/
```

3. 将上述三个文件放入该目录。

4. 重启 Obsidian。

5. 打开：

```text
设置 → 第三方插件 → Cloze Plus
```

6. 启用插件。
